# Sitemap_generator
Скрипт для генерации sitemap'инга ;)
Многопоточная реализация*

На вход:
- Файл input.txt с сылками сайтов (новая ссылка- новая строка)

На выход:
- Файлы sitemap_(link).xml
- Пикча result.png с итоговыми результатами работы скрипта
